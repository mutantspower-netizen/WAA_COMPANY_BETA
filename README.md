# Company Website Beta Prototype

## Project Overview
This project was developed during my internship as part of the UX/UI and website implementation process. The website prototype was designed to improve user experience, navigation consistency, and responsive accessibility across different devices.

## Features
- Responsive website layout
- Consistent navigation bar design
- Dropdown menu implementation
- Mobile-friendly interface
- Improved content organization

## Tools and Technologies
- React.js
- Builder.io
- Tailwind CSS
- Figma
- Netlify

## Installation and Startup

1. Clone or download the project files
2. Open the project folder
3. Install dependencies

```bash
npm install
```
```bash
npm run dev
```
